Qsee is a free, on-prem, website-based Quality Control tool for text data.

## What Qsee Does
- Upload Excel and JSON files
- Edit text directly in the browser
- Merge datasets (mandatory step)
- Run text quality checks
- Review results clearly and safely

## Core Quality Checks
- Grammar & spelling (English & Korean)
- Duplication detection
- Plagiarism detection
- AI-generated text detection

## Key Principles
- Free to use
- Fully on-premises
- No data leaves your system
- Built for human review and trust
